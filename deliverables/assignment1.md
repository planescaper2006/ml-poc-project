Mon projet : Mon projet consiste à développer un modèle de machine learning permettant de prédire l’issue d’un match de League of Legends (victoire ou défaite) à partir des statistiques du début de partie.

L’objectif est d’identifier les facteurs clés qui influencent le résultat d’un match et d’évaluer la capacité d’un modèle à anticiper l’issue dès les premières minutes de jeu.

Le business case: Dans les jeux compétitifs comme League of Legends, la capacité à analyser rapidement les performances et à comprendre les facteurs de victoire est essentielle.

Ce projet s’inscrit dans une logique d’analyse de données appliquée à l’esport. Il vise à fournir des insights permettant :

* d’identifier les éléments déterminants d’une victoire
* d’améliorer la prise de décision en cours de partie
* d’aider à l’analyse stratégique des équipes

Ce type de modèle peut être utilisé par des équipes esport, des analystes de données ou des plateformes spécialisées pour optimiser les performances et mieux comprendre le jeu.

les sources données : API riot games
